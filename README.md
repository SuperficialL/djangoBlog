# djangoBlog

## Project setup
```djangotemplate
# 安装所需要的包
pip install -r requirements.txt
```
```mysql
# 创建数据库
CREATE DATABASE `website` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci;

```
```
# 创建迁移文件
python manage.py makemigrations  
# 生成数据表结构
python manage.py migrate
# 服务器运行
python manage.py runserver  

```
